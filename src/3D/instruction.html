msViz Graph Help

Navigation

  • Pan: Left drag; arrow keys
  • Zoom: Ctrl + left drag; scroll wheel
  • Rotate: Right click
  • 2D perspective: Hold Shift

Toolbar Buttons

The open button prompts for a compatible mass spec data file.
The save as button makes a new copy of the current dataset.

Toggles between a 3-dimensional view and a top-down 2-dimensional view.
Refreshes data from the server.
Zooms the graph out as far as possible, showing all data points in the set.
Adjust the number of points visible on screen.
Select colorblind mode to switch the palette used for data points.
The sidebar button tooltips can be turned off if they are in the way.
